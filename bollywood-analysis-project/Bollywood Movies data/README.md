A SQL project analyzing what drives the financial success of Bollywood films using a publicly available dataset.

Overview

This project explores box office performance, profitability, remakes, franchises, and screen distribution using core SQL queries.
The goal is to practice and demonstrate strong SQL fundamentals while deriving simple business insights from movie data.

Dataset

The dataset includes information such as:

Movie Name

Genre

Release Period

Lead Star

Director

Number of Screens

Revenue (INR)

Budget (INR)

Whether the film is a remake or franchise


Analysis Performed

Key analyses include:

Top revenue-generating films

Movies earning more than 2× their budget

Revenue comparison of remakes vs originals

Average revenue of franchise films

Directors with multiple films

Identification of loss-making movies

Profit calculation (revenue - budget)
